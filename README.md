Companies Directory – Frontend Assessment

A responsive and user-friendly React (Vite) application that displays a directory of companies with search, filters, sorting, and dual layout views. This project was developed as part of a technical assessment to demonstrate frontend development skills, UI design, and API integration.

Live Demo
Vercel Deployment: https://companies-directory-u99v.vercel.app/

Features
Core Requirements
• React-based responsive UI
• List of companies displayed in Table View
• Card View layout option
• Live search (company, industry, location)
• Filter by industry
• Filter by location
• Sorting by Name (A to Z / Z to A)
• API integration using a mock endpoint (public/companies.json)
• Clean UI with proper spacing and accessibility

Tech Stack
• React + Vite
• JavaScript (ES6+)
• CSS (custom styling)
• Mock API (JSON file)
• Vercel for deployment
